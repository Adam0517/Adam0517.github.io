Introduction : Latitude vs. Weather factors


The picture is from this website

The purpose of this topice is visualizing the weather of 500+ cites acroos the world of varying distance from the equator. We first used OpenWeatherMap API to get weather datasets over 500 cities.


Then, we used Python Matplotlib to plot various aspects of latitude vs. weather factors (temperature, cloudiness, wind speed, and humidity). This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
